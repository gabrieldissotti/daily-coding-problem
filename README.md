# Daily Coding Problem

Creating algorithms with typescript

## Setup

Just run yarn

## How To run

```sh
# Run script
nodemon ./problems/{number}/index.ts
# Example:
nodemon ./problems/1/index.ts

# Run all tests
yarn test

# Run specific test example
yarn jest ./problems/1/index.test.ts
```
